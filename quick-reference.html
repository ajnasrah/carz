STEP-BY-STEP: EVALUATING & BUYING A VEHICLE

1 Find Vehicle on OVE
Note: VIN, Year/Make/Model, Mileage, CR Grade, Buy Now price
2 Pull Autoniq Report
Enter VIN → Get Black Book History Adjusted AVG
Note: 2WD/RWD = LOW DEMAND. Don't stretch on price!
3 FIRST CHECK: Is there a $3,000 spread?
Book Value − Buy Now Price ≥ $3,000?
NO → SKIP THIS CAR. Don't waste time. Move on.
YES → Continue to Step 4
4 Check History Reports
Review AutoCheck for accidents, owners, damage
STOP if: Frame damage, Flood, Salvage title
5 Read the CR (Condition Report) - EVERY LINE
Note all issues: mechanical, body, interior, tires, announcements
6 Estimate Rehab Costs
Add up all repairs from CR
For major parts: Check Car-Part.com
Target: Under $3,000 total rehab
7 FINAL MAX BID = Book Value − $3,000 − Rehab
The $3,000 covers fees + profit. Rehab is EXTRA on top.
8 Compare to Buy Now Price
Buy Now ≤ Max Bid? → BUY IT
Buy Now > Max Bid? → PASS

THE MAX BID FORMULA

Max Bid = Average Book Value - $3,000 - Rehab Costs

Example: $25,000 book - $3,000 spread - $1,500 rehab = $20,500 Max Bid

HOW TO FIND "WHAT IT WILL SELL FOR" (Autoniq)

Pull Autoniq for the vehicle. Use the Black Book AVERAGE as your baseline:

Quick Rules:
  • CR Grade 3.0+: Use Black Book Average
  • CR Grade 2.0-2.9: Use Black Book Average or between Avg/Rough
  • CR Grade 1.0-1.9: Use Black Book Rough
DEMAND IS #1: Book value already adjusts for 2WD vs 4WD. The real difference is how fast it sells. Target HIGH DEMAND configs!
